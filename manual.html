RHA Workbook Manual

This manual provides agents with the necessary guidelines to use the RHA Workbook effectively. Our goal is to maintain a centralized, real-time knowledge base for the entire team.

1. How to Navigate

The Menu ☰ button at the top left is your primary tool for switching between modules:

  • Self-Pay: Access current pricing for patients without insurance.
  • Clinic Hours: Real-time updates on provider availability.
  • Visit Types: Descriptions of appointment categories.

2. Using the Live Notepad (Hub)

The Notepad is a live Google Document. Multiple agents can type at once.

  • Auto-Save: You do not need to click save; changes sync instantly.
  • Formatting: Use the built-in toolbar for bolding or highlighting urgent updates.
  • Etiquette: Avoid overwriting notes from other agents without prior discussion.

3. Admin & Support

If you encounter broken links or "Access Denied" errors, please head to the Admin Control page to submit a ticket or contact your supervisor directly.
